Requirements

Get_Check_Total

Get_Check_Total.1
The system shall return the check total associated with the table record identified by the provided table index.

Get_Next_Party_To_Be_Seated

Get_Next_Party_To_Be_Seated.1
The system shall reset the waiting list index to 0 if the index is greater than 9 before accessing the waiting list.
Get_Next_Party_To_Be_Seated.2
The system shall not modify the waiting list index if the index is 9 or less before accessing the waiting list.
Get_Next_Party_To_Be_Seated.3
The system shall return the party at the current waiting list index and increment the index after returning the party.

Add_Included_Dessert

Add_Included_Dessert.1
The system shall set the dessert to PIE when the entree is STEAK, the salad is CAESAR, and the beverage is MIXED_DRINK.
Add_Included_Dessert.2
The system shall set the dessert to CAKE when the entree is LOBSTER, the salad is GREEN, and the beverage is WINE.
Add_Included_Dessert.3
The system shall not modify the dessert when the entree, salad, and beverage do not match either (STEAK, CAESAR, MIXED_DRINK) or (LOBSTER, GREEN, WINE).

Clear_Table

Clear_Table.1
The system shall set the table's occupied status to false and the number in party to one when clearing a table.
Clear_Table.2
The system shall set all order fields for each seat at the table to their respective 'no' values when clearing a table.
Clear_Table.3
The system shall reset the order fields for exactly four seats when clearing a table.
Clear_Table.4
The system shall set the check total to zero, update the table record, and return zero when clearing a table.

Add_Party_To_Waiting_List

Add_Party_To_Waiting_List.1
The system shall reset the waiting list index to zero when the waiting list is full before adding a new entry.
Add_Party_To_Waiting_List.2
The system shall retain the current waiting list index when the waiting list is not full.
Add_Party_To_Waiting_List.3
The system shall copy each character from the provided name into the next available waiting list entry when a non-empty name is provided.
Add_Party_To_Waiting_List.4
The system shall not copy any characters into the waiting list entry when the provided name is null or empty.
Add_Party_To_Waiting_List.5
The system shall null-terminate the new waiting list entry and increment the waiting list size after adding a name.

Place_Order

Place_Order.1
The system shall retrieve the table record, mark the table as occupied, increment the party size, store the order for the specified seat, and add an included dessert to the order if the order qualifies.
Place_Order.2
The system shall not change the check total when the order's entree is 'no entree'.
Place_Order.3
The system shall add 14.0 to the check total when the order's entree is steak.
Place_Order.4
The system shall add 10.0 to the check total when the order's entree is chicken.
Place_Order.5
The system shall add 18.0 to the check total when the order's entree is lobster.
Place_Order.6
The system shall add 12.0 to the check total when the order's entree is pasta.
Place_Order.7
The system shall not change the check total when the order's entree does not match any defined entree value.
Place_Order.8
The system shall update the table record with the new data and return a success indicator.
